## Run Apache Kafka docker container
```
docker compose up

```

### Create new topic

// docker compose initial 2 topic my-topic-1 & my-topic-2



## Start NodeJs App

```
npm i

node index.js
```


## Send Message to Apache Kafka

```
curl --location 'http://localhost:3000/send' \
--header 'Content-Type: application/json' \
--data '{"message": "Hello, Apache Kafka!"}'
```
